FRC Strategic Dashboard

Welcome to the FRC Strategic Dashboard. This application provides real-time analytics and predictions for FRC matches.

API Information

The backend API is running. You can access the following endpoints:

/api/v1/matches

Get match data and results

/api/v1/teams

Get team information and statistics

/api/v1/predictions

Get match predictions using ML models

/api/v1/stats

Get statistical analysis

API Documentation

For detailed API documentation, visit: /api/docs

For interactive API exploration, visit: /api/redoc

Frontend Status

The React frontend is currently being built. Once complete, it will be automatically served from this endpoint.

If you see this page, please run cd frontend && npm install && npm run build to build the frontend.
